Projetos Recentes

Ambiente virtual adaptado para MADDPG

O projeto consiste na adaptação de um ambiente de simulação de futebol de robôs para o estudo de aprendizado por reforço multi-agente. A base do trabalho foi o framework rSoccer, que utiliza o simulador robosim para recriar as dinâmicas da competição de robôs de Very Small Size Soccer (VSSS). Para viabilizar o treinamento, o ambiente foi reestruturado para ser compatível com o algoritmo Multi-Agent Deep Deterministic Policy Gradient (MADDPG) e a API PettingZoo, que permite a gestão e a interação individualizada com múltiplos agentes através de uma estrutura de dados baseada em dicionários.
